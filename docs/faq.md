# Frequently Asked Questions

Welcome to the Tempo FAQ.  
Here we try and answer any question that comes up frequently.  
If you can't find what you are looking for here, see our other [help](./help.md) channels.  


## What platforms does Tempo support?

Currently we support Windows (x86-64 and i686) and Linux (x86-64 and aarch64).

---

## As a game developer, can I use this for my game's mod support?

Yes, the license fully allows this, and we would love to see it.
